Cognitive Voice Gateway
WebRTC Reference Client

The following button is provided by a Web Component. It receives all the necessary information, in order to make a SIP call using WebRTC, from the form above. Attributes and lifecycle events on the Web Component allow for customization of the displayed controls during the call.

Have a look at the Source Code on GitHub and specifically the files index.html, index.css and example.ts.

If you don't have the information necessary to fill out the form and start a call, don't hesitate to get in touch with our support at support@vier.ai.

In case of technical issues with this application please report issues on GitHub. Contributions through Pull Requests are welcome!
